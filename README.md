# formatter314
Java Autoformatter modelled after the code style guide provided in the CS 314 course at UT Austin.

Compilation:
```sh
git clone https://github.com/zarbod/formatter314/ && cd formatter314
cargo build --release
```
the binary will be stored in mike_formatter/release

Usage:
```
./formatter314 Example0.java Example1.java Example2.java
```

Full formatting specification will be added in time. 
Features listed in features.md
